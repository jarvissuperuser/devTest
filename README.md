# Chat App
## Description
SPA Chat app prototype

##How to setup
### Dependencies

please make sure the `node` and `npm` are install


this app requires node version 8 LTS to run

    $ node --version
    8.x.x
    $ npm --version
    6.x.x
    
### Launching the app

    $ npm start

## Routes
   
the apps routes >> [localhost:3000/home.html](http://localhost:3000/home.html)
